MyFlix Movie API Documentation

MyFlix is a web application that provides users with access to information about different movies, directors and genres.
Users can register, update their personal information and create a list of their favorite movies.

Request URL HTTP Method Query Parameters Request body data format Response body data format

Return information about the user /user/:Username GET none none A JSON array of objects containing the informacion of the user
Allow users to update their information / users/:Username PUT :Username A JSON object containing the user's data to be updated like:
{
username: "String",
password: "String" ,
email: "String",
Birthday: "Date"
}
 A JSON object holding data about the user's updated information
Allow users to add a movie to their favorites list / users/:Username/movies/:MovieID POST :Username :MovieID none A JSON object holding data about the user's updated information (MovieID added)
Allow users to remove a movie from their favorites list /users/:Username/movies/:MovieID DELETE :Username, :MovieID none A JSON object holding data about the user's updated information (MovieID deleted)
Allow users to delete their account / users/:Username DELETE :Username none A text message indicating that the user's account has been deleted.
Return a list of All movies to the user /movies GET none none A JSON array of objects containing the informacion of all the movies
Return data about a single movie by title to the user /movies/:Title GET :Title none A JSON object containing information about the requested movie
Return data about a genre (description) to the user /movies/genre/:genreName GET :genreName none A JSON object containing information about the requested genre
Return data about a director (bio , birth year) to the user /movies/directors/:directorName GET :directorName none A JSON object containing information about the requested director
