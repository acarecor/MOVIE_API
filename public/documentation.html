My Flix Documentation

MyFlix is a web application that provides users with access to information about different movies, directors and genres.
Users can register, update their personal information and create a list of their favorite movies.

Request URL HTTP Method Query Parameters Request body data format Response body data format
Allow new users to create an account /users POST none A JSON Object holding data about the user to be added, structured like:
{
name: "Julian Smith",
favoritesMovies: [ "Citizen Kane"]
} 		A JSON object containing data about the user that has been added, including an ID:
{
id:1,
name: "Julian Smith",
favoritesMovies: [ "Citizen Kane"]
}
Allow users to update their information (username) / users/[id] PUT :id A JSON object containing the user's data to be updated like:
{
id:1,
name: "Jul Smith",
favoritesMovies: [ "Citizen Kane"]
}
 A JSON object holding data about the user's updated information like:
{
id:1,
name: "Jul Smith",
favoritesMovies: [ "Citizen Kane"]
}
Allow users to add a movie to their favorites list / users/[id]/[movieTitle] POST :id, :movieTitle none A text message indicating the title of the movie that has been added to favorites.
Allow users to read the list of their favorite movies /users/[id]/favorites GET :id none a JSON array containing the list of favorite movies from the user.
Allow users to remove a movie from their favorites list /users/[id]/[movieTitle] DELETE :id, :movieTitle none A text message indicating the name of the movie that has been removed from favorites
Allow users to delete their account / users/[id] DELETE :id none A text message indicating that the user's account has been deleted.
Return a list of All movies to the user /movies GET none none A JSON array of objects containing the informacion of all the movies
Return data about a single movie by title to the user /movies/[title] GET :title none A JSON object containing information about the requested movie
Return data about a genre (description) to the user /movies/genre/[genreName] GET :genreName none A JSON object containing information about the requested genre
Return data about a director (bio , birth year) to the user /movies/director/[directorName] GET :directorName none A JSON object containing information about the requested director
